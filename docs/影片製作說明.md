# 影片製作說明（Production Guide）

這支教學影片是**完全用程式產生**的：動畫投影片用瀏覽器播放並錄影，
實際操作畫面用 Playwright 驅動真實的 JupyterLab 錄製，最後用 ffmpeg 加上溶接轉場與淡出，輸出成 MP4。

> 成品：`video_build/output/tutorial_final.mp4`（約 3:18，1920×1080，30fps）

---

## 🧩 用到的工具（皆在環境內）

| 工具 | 用途 |
|------|------|
| **Playwright（Chromium）** | 播放動畫投影片並**錄製成影片**；驅動真實 JupyterLab 錄製操作畫面 |
| **ffmpeg**（`ffmpeg-static`） | 正規化、溶接轉場（xfade）、淡出、輸出 MP4 |
| **JupyterLab ＋ nbconvert** | 執行 Notebook、產生**真實圖表**、提供實際操作畫面 |
| **Python（scikit-learn / xgboost / matplotlib）** | 用虛構資料跑出真實的分析結果 |

---

## 📁 檔案結構

```text
video_build/
├── deck/index.html        # 動畫投影片（所有文字、轉場、字幕都在這）
│   └── assets/            # 投影片裡用到的真實圖表（由 Notebook 產生）
├── record_deck.mjs        # 錄製投影片：node record_deck.mjs <名稱> <起始scene> <結束scene>
├── record_jupyter.mjs     # 錄製真實 JupyterLab 操作畫面
├── shot.mjs               # 擷取單一 scene 靜態截圖（除錯／做縮圖用）
├── assemble.sh            # 把所有片段用 ffmpeg 溶接 + 淡出，輸出最終 MP4
├── clips/                 # 各段原始錄影（.webm）
└── output/tutorial_final.mp4
```

## 🔁 如何重新產生整支影片

```bash
# 1) （如尚未產生）跑出真實圖表
cd tutorial_demo/Data && python make_synthetic_data.py && cd ..
cd Notebooks && for n in 1_Feature_Selection 2_Model_Development 3_External_Validation; do \
  jupyter nbconvert --to notebook --execute --inplace $n.ipynb; done
cp ../Figures/*.png ../../video_build/deck/assets/

# 2) 啟動真實 JupyterLab（給實際操作畫面用）
cd ../../tutorial_demo
jupyter lab --no-browser --port=8899 --ip=127.0.0.1 \
  --ServerApp.token='' --ServerApp.disable_check_xsrf=True --allow-root \
  --ServerApp.root_dir="$(pwd)" &

# 3) 錄製三段
cd ../video_build
node record_deck.mjs deck_a 0 12     # 片頭 → LIVE DEMO 過場
node record_jupyter.mjs              # 真實 JupyterLab 操作
node record_deck.mjs deck_c 13 17    # Manuscript → 片尾

# 4) 合成（溶接轉場 + 淡出）
bash assemble.sh
```

## ✏️ 如何客製化

- **改文字／字幕／旁白點：** 直接編輯 `deck/index.html`
  - 投影片文字在各 `<section class="scene">` 內。
  - 螢幕下方字幕在 `CAPS` 陣列（每個 scene 一句，順序對應）。
  - 每個 scene 的停留秒數＝ `data-dur`（毫秒）。
- **改配色／字體：** `deck/index.html` 最上面的 `:root{ --accent … }`。
- **加長／縮短某段：** 改該 scene 的 `data-dur`，重錄該段即可。

### 換成你自己的「真實操作」畫面
影片在 **STEP 3（與 AI 溝通）** 與 **LIVE DEMO** 兩處，最適合放你自己的實拍：
1. 自行用螢幕錄影軟體（OBS、QuickTime…）錄下你**真實操作的畫面**，例如：
   - 在 **Google Colab** 跑 Notebook
   - 用 **Claude Code** 等 AI Agent 對話、請它寫 Notebook
   - 在 **GitHub** 上建立 Repo、push 程式碼
2. 把錄好的檔案放進 `clips/`，命名如 `clips/colab.webm`。
3. 在 `assemble.sh` 的 xfade 鏈中，把對應片段換成你的檔案、調整 offset。

> ⚠️ 用真實帳號錄影前，請先確認畫面中**沒有任何真實病人資料**或可識別資訊。

## 🔊 關於配音

目前環境沒有語音合成（TTS），因此影片為**內嵌字幕的靜音版**。
請依 `docs/教學影片_旁白與分鏡腳本.md` 的旁白稿自行配音，再於剪輯軟體中對齊。
（若要自動配音，可改用線上 TTS 服務產生 mp3，再用 ffmpeg 以 `-i audio.mp3 -c:a aac` 併入。）
